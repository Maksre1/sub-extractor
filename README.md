# SubExtractor

Удобный, конфиденциальный и современный веб-сервис для извлечения VPN-ключей (VLESS, Hysteria2 и др.) из подписок различных клиентов (Happ, Incy, Hiddify, Clash, а также стандартных Base64 и текстовых списков).

Разработано на Next.js в строгом минималистичном SaaS-стиле.

## Основной функционал

*   **🔒 Локальная конфиденциальность**: Все данные обрабатываются в оперативной памяти и нигде не сохраняются. Генерация QR-кодов происходит полностью на стороне клиента в браузере (без обращения к сторонним API).
*   **🔌 Поддержка множества форматов**:
    *   Ссылки приложений Happ, Incy, Hiddify, FoxTunnel (включая редиректы `happ://add/`, `incy://import/`).
    *   Стандартные подписки V2Ray/Xray в кодировке Base64.
    *   Сырые списки ссылок (vless, hysteria2, trojan, vmess и др.).
*   **⚡ Встроенный тест задержки (Ping)**:
    *   Замер пинга на стороне сервера с умным обходом ограничений UDP (Hysteria2) — автоматически переключается на доступные порты VPS (SSH/22 или Web/80) для точного замера сетевого RTT.
    *   Индикация пинга с цветовой маркировкой.
*   **📱 QR-коды для импорта**:
    *   Генерация QR-кода на HTML5 Canvas для быстрого сканирования камерой мобильного устройства.

## Как запустить локально

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Откройте страницу [http://localhost:3000](http://localhost:3000) в браузере.

## Развертывание (Deploy)

Проект полностью оптимизирован для быстрого деплоя на платформу **Vercel** в один клик через импорт репозитория GitHub.
